# My Setup

---

## Contributing

### Getting started

Clone the project:

```bash
git clone git@github.com:Les-Cop1/my-setup-back.git
```

Add environment variables:

```bash
cp .env.example .env
```

Set up the project:

```bash
yarn setup
```

### Development

To start the development server you need to run the following commands:

```bash
yarn start:development
```
